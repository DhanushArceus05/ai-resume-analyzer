# 🚀 AI Resume Analyzer

An AI-powered Resume Analyzer built with **React, TypeScript, Node.js, Express, MongoDB Atlas, and Google Gemini AI**. It helps job seekers analyze resumes, calculate ATS scores, match resumes against job descriptions, generate interview questions, rewrite resume content, and export professional PDF reports.

---

## 🌐 Live Demo

**Frontend:** https://ai-resume-analyzer-version-one.vercel.app

**Backend API:** https://ai-resume-analyzer-e67b.onrender.com

# ⚙️ Local Installation

## Clone Repository

```bash
git clone https://github.com/DhanushArceus05/ai-resume-analyzer.git
```

```
cd ai-resume-analyzer
```

---

## Backend

```bash
cd backend
npm install
```

Create `.env`

```env
MONGODB_URI=your_mongodb_uri

JWT_SECRET=your_jwt_secret

GEMINI_API_KEY=your_gemini_api_key

CLIENT_URL=http://localhost:5173
```

Run

```bash
npm run dev
```

---

## Frontend

```bash
cd frontend
npm install
```

Create `.env`

```env
VITE_API_BASE_URL=http://localhost:5000/api
```

Run

```bash
npm run dev
```
